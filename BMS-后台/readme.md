# Nodejs项目

## 后台管理系统
>介绍：在二阶段项目的基础上实现一个电商网的后台管理系统

### 项目情况-运用技术
* 后端语言：使用的是Nodejs+Express
* 数据库：使用的是MongoDB
* UI框架：使用bootstrap框架
* 登陆使用token进行加密、解密、验证
* 其他页面使用nodejs+Express和其他第三方插件
